# sprint 브랜치로 들어가기

1. 원하는 곳에 git clone
```zsh
git clone https://yeardream-gitlab.elice.io/yeardream-project/project-9/service.git
```

2. 클론된 폴더로 이동 & sprint 브랜치로 변경
```zsh
cd service
git checkout sprint
(안되면) git chechout -t origin/remote
```
